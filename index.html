SASS

Syntactically Awesome Style Sheets

It turns this:


 $brand-color: hsl(207,41,75);
 $font-size: 1.3rem;
 $padding: 1rem;
 $font: #{$font-size}/#{$font-size * 2} Arial;
 
 body {
   background-color: $brand-color;
   font: $font;
 }
 section {
   padding: $padding;
   background: lighten($brand-color, 20%);
   h2{
     text-transform: uppercase;
   }
   p{
    
   }
 }

Into this:


 body{
   background-color: #A5C2D9;
   font: 1.3rem/2.6rem Arial;
 }
 section{
   padding: 1rem;
 }
 section h2{
  text-transform: uppercase;
 }
